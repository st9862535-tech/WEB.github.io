# WEB.github.io

Проєкт присвячений розробці вебсайту для оренди автомобілів.
Проєкт повністю реалізував відповідно до данного макету (https://www.figma.com/community/file/1381749952787054469)
Але під час завершального етапу роботи з CSS сталося відключення світла, я працював у Notepad++,
і там немає резервного копіювання, У результаті зараз некоректно відображаються сторінки About Us /Contact Us,(Зламалася структура CSS)
мені не вдається знайти та виправити помилку через складність коду.

На данний момент працюють сторінки Home, Vehicle, Details(Деталі про авто при кліку "ДЕТАЛЬНІШЕ")
В той момент після перезапуску(Вибило світло) у мене залишилися відкриті вкладки сторінок About Us, Contact Us з правильним відображенням та я записав відео яке це підтверджує. (Цих двох сторінок, завантажив [на youtube.com](https://youtu.be/scNdxezCnPk)
